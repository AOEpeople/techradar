For new microservices in a cloud environment, Spring Boot is still one of our defaults. Apart from the before mentioned reasons, we at AOE like especially the following features
- full integration with the [Kotlin language](/techradar/languages-and-frameworks/kotlin/)
- excellent plugins and libraries for supporting GraphQL ([DGS Framework](https://netflix.github.io/dgs/), [Spring for GrqphQL](https://spring.io/projects/spring-graphql))
- easy to use and powerful provision of RESTful services including support for OpenAPI
- support for many cache providers, including distributed and replicated caches
- metrics and monitoring support with [Micrometer](https://micrometer.io/)
